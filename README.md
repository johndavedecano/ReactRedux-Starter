##Technology Stack

1. NodeJS
2. ReactJS + Redux
3. ExpressJS
4. WebPack

##Installation
  1. Clone repository git clone https://github.com/johndavedecano/api-org-app.git
  2. Configure your mongoDB database URI and credentials at config.js.

```
      # Install npm packages
      npm install
      # Build Application
      npm run build-production
      # Fire up the server
      npm start
```
